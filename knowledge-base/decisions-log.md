# Decisions Log — the learning loop

> Every time Adam reacts to a role ("yes, that fits" / "no, not that"), record it here.
> Before each new `/find-roles` sweep, read this file — it sharpens targeting over time.
> `/log-decision` appends entries; `/refine-search` rolls patterns up into `fit-criteria.md`.

## Format
```
### YYYY-MM-DD — [Firm / Role or pattern]
- **Verdict:** FIT / NO-FIT / MAYBE
- **What Adam said:** (his actual words/reaction)
- **Why:** (the underlying signal)
- **Criteria implication:** (what to weight up/down, any new gate)
```
